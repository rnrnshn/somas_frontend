
  # App Design from Document (Copy)

  This is a code bundle for App Design from Document (Copy). The original project is available at https://www.figma.com/design/Qfn2BGr04VV6rgyPdiOXdD/App-Design-from-Document--Copy-.

## Running the code

Use `pnpm install` to install the dependencies.

The app uses `VITE_API_BASE_URL` and is configured by default to point at the same API as `somas_frontend`:

`https://api.somas.arrentis.co`

Run `pnpm dev` to start the development server.
