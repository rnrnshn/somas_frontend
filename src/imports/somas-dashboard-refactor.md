Refactor Dashboard Typography and KPI Cards – SOMAS

Refactor the dashboard layout and KPI cards for the SOMAS backoffice interface to create a clean, minimal, and data-focused design.

The interface should feel calm, professional, and institutional.

Remove visual clutter and simplify the presentation of metrics.

Follow Shadcn design system principles and maintain a restrained visual style suitable for an operational platform.

⸻

Typography

Use the following typography system.

Font Family

Avoid large or oversized typography.

The interface should feel subtle and balanced.

⸻

KPI Cards

Simplify all KPI cards.

Remove all icons from KPI cards.

Each KPI card must contain only:

KPI label
Primary value
Optional supporting text

Example layout

Total Disbursed
3 400 000 MZN
Last 30 days

The number must be the visual focus.

⸻

Card Style

Use a simple card layout.

Background: white
Border: light neutral border
No heavy shadows
No gradients
No colored backgrounds

Use comfortable padding and subtle spacing.

⸻

Icons Usage

Icons should be used only where they improve usability.

Use icons only for:

Sidebar navigation
Primary actions such as search, export, retry, sync
Status indicators where necessary

Do not use icons inside KPI cards.

⸻

Spacing

Use consistent spacing:

8 px
16 px
24 px
32 px

Increase white space between dashboard sections to reduce visual density.

⸻

Grid Layout

KPI cards should be arranged in a simple grid.

Display:

4 cards per row on large screens
3 cards per row on medium screens

Cards should be compact and aligned.

⸻

Visual Tone

The dashboard must feel:

Minimal
Calm
Professional
Institutional

Avoid a colorful or marketing-style dashboard.

The focus must remain on data clarity and readability.
