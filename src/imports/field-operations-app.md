Field App – SOMAS

Design the Field Operations mobile app for the platform SOMAS – Social Mass Payments & Savings Platform.

The Field App is used by field enumerators to verify beneficiaries participating in social transfer programs.

The mobile app must be aligned with the backoffice system and allow enumerators to search for beneficiaries, verify their identity, and capture verification evidence.

The design should be simple, mobile-friendly, and optimized for field operations.

Follow the existing design system and maintain a clean, minimal interface.

Create the following screens.

Login Screen

Allow enumerators to log in using email and password.

Display the SOMAS logo and a simple login form.

⸻

Home Dashboard

Display operational information for the enumerator.

Include cards showing:

Assigned Beneficiaries
Verifications Completed Today
Pending Sync Records

Include main actions:

Search Beneficiary
My Verifications
Sync Data

⸻

Search Beneficiary

Allow searching by:

Beneficiary Name
Phone Number
Beneficiary Code

Display results in a list showing:

Name
Province
Campaign
Verification Status

⸻

Beneficiary Profile

Display detailed beneficiary information.

Include:

Full Name
Phone Number
Province
District
Campaign
Verification Status

Include a button to start verification.

⸻

Verification Screen

Allow the enumerator to confirm beneficiary details.

Include fields for:

Confirm Identity
Confirm Phone Number
Enumerator Notes

Include a button to capture a selfie.

⸻

Selfie Capture

Allow the enumerator to capture a photo of the beneficiary as verification evidence.

Automatically capture:

Timestamp
GPS Location
Enumerator ID

⸻

Submit Verification

Allow the enumerator to submit the verification result.

Verification status options:

Verified
Rejected
Pending

⸻

Offline Sync

Display records waiting to be synchronized with the server.

Include:

Pending Verifications
Last Sync Time
Sync Button

⸻

Daily Summary

Display operational statistics for the enumerator.

Include:

Total Verifications
Successful Verifications
Pending Sync Records
