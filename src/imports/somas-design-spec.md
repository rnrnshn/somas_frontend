Create a new product design from scratch for:

SOMAS – Social Mass Payments & Savings Platform

SOMAS is a government-aligned, compliance-driven financial infrastructure platform used for large-scale social transfer operations.

The platform includes two environments:
	1.	Backoffice Web Platform (Admin, Monitoring, Reporting)
	2.	Field Operations Mobile App (Enumerators)

Start clean.
Start structured.
Start scalable.

⸻

CORE DESIGN OBJECTIVE

Design a simple, elegant, institutional interface using Shadcn Design System principles.

The interface must feel:
	•	Professional
	•	Calm
	•	Structured
	•	Modern
	•	Minimal
	•	Trustworthy
	•	Audit-ready

Avoid:
	•	Visual clutter
	•	Decorative gradients
	•	Startup-style UI
	•	Overuse of color
	•	Heavy shadows

Elegance through restraint.

⸻

DESIGN SYSTEM FOUNDATION (MANDATORY)

Follow Shadcn UI structure:
	•	Clean spacing scale
	•	Subtle borders
	•	Muted backgrounds
	•	Consistent variants
	•	Clear typography hierarchy
	•	Accessible states
	•	Developer-friendly component logic

Prefer border separation over heavy elevation.

⸻

GLOBAL STYLE SETTINGS

Typography:
Inter

Type Scale:
	•	H1: 32px
	•	H2: 24px
	•	H3: 18px
	•	Body: 14–16px
	•	Caption: 12px

Spacing scale:
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64

Border radius:
Medium (rounded-lg equivalent)

Shadows:
Minimal. Prefer subtle borders.
